# TYPESCRIPT

- [ ]  run `npm init`
- [ ]  Create file `index.html`.
- [ ]  Create folder `src`.
- [ ]  Create file `src/index.ts`.
- [ ]  Run `npm i typescript --save-dev`
- [ ]  run `tsc --init`
